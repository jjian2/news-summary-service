# 📰 뉴스 요약 웹 서비스

## 📌 프로젝트 정보
- 기간 : 2025.03 ~ 2025.05  
- 기여도 : 100%

## 📖 프로젝트 소개
뉴스 기사 URL을 입력하면 자동으로 본문을 수집하고 자연어 처리 모델을 통해 요약 결과를 제공하는 웹 서비스입니다.

## 🧑‍💻 담당 역할
- Flask 기반 요약 API 서버 구축  
- HuggingFace 요약 모델 연동  
- JSP와 API 통신 구조 설계  

## 📚 Lesson Learned
- CORS 및 인코딩 문제 해결 경험  
- AI 모델 API화 경험  
- 프론트엔드-백엔드 통합 흐름 이해
